# s21_decimal — точная десятичная арифметика

Реализация типа `decimal` для финансовых расчётов без ошибок округления.

## Особенности
- Точность до 28 знаков после запятой
- Диапазон: ±79 квинтиллионов
- Поддержка: +, -, *, /, сравнение, преобразования
- Безопасное округление (банковское)

## Сборка
```bash
make          # Собрать библиотеку
make test     # Запустить тесты
make gcov_report  # Создать HTML-отчёт покрытия
make clean    # Очистить
